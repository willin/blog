# 安装

- Github [下载插件](https://github.com/Authing/wp-authing/releases) 并上传安装
- 插件安装页面搜索 Authing 并安装

![Config](/images/blog/authing-wordpress/wp-authing-config.png)

按照插件提示进行配置。

- 填写 Authing 应用域名
- 将对应的配置项在 Authing 控制台中进行设置
- 填写 App ID 和 App Secret 进行保存

Authing 控制台配置示意如下图：

![Config](/images/blog/authing-wordpress/authing-config.png)

---

配置完成后可以注销，重新登录。

![Login](/images/blog/authing-wordpress/wp-login.png)

如果已经呈现上图所示，说明配置成功。
