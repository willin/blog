## 程序思想

自定义游戏规则：

- 4x4 格子
- 操作只能：上下左右 4 个动作
- 空白处（非碰撞）随机出现一个 2 或 4
- 操作一个方向进行相同数字合并相加
- 游戏开局：随机放置两个数字，2 或 4
- 失败条件：所有格子满了，并且不能操作合并
- 胜利条件：当最大数字达到 2048 时胜利
- 计分：屏幕上所有数字之和
- 开始按钮：重新开局
- 历史最高分：持久记录，如果当前积分大于历史最高则更新

通过 CSS + HTML 先将游戏界面完整实现，然后再通过 JavaScript 来实现游戏逻辑。
