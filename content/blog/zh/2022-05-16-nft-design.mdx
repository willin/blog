# 平台选择

国内平台，如火链基础支撑的， Mars 星云、秦宇宙等（可以根据项目和需要引荐国内不同的平台）。

国外平台，如 Opensea 等基于 ETH 的平台，以及其他币种的平台。该如何选择，成本控制，以及其他常见问题解答。

# 前期规划

技术角度来说至少包括部件的种类该如何设计，各个部件的概率该如何设计，这过程中会有什么样的坑。

商业角度来看定什么样的计划，比如一共发行多少张，其中空投多少比例，怎么样运作使利益更大化等。

以及其他可能出现的风险和问题，需要准备的工作和资源。

# NFT 设计

设计过程中应该怎么样进行效果自测，怎样能高效地设计出有用的组件素材。

素材的规范，比如图片的格式尺寸要求，内容形式要求，注意点。

如何在设计过程中避免可能出现返工的问题等。

# 批量生成

- 如何针对概率设计生成合成素材表格
- 如何批量化生成 NFT 图片产物
- 如何针对不同的类型（如性别）样式进行区分
- 如何批量进行图片压缩、重命名等优化工作

其他技术相关技巧和其他常见的问题处理方法等。

# 平台上线

在平台上线之前，如何将数据和图片进行关联，数据结构的设计等。

空投、预售、公售相关常见问题等。
